Be the Hero é uma aplicação web e mobile para ONGs animais se cadastrarem, postarem seus casos e valores e receberem doações.
O Projeto foi criado na "Semana OmniStack 11.0"
Ela foi construida utilizando:
Node.js; 
SQLite3;
Knex;
Axios;
Nodemon;
ReactJS;
React Native;
Celebrate;
Joi;
Jest;
Supertest;
